# create

### Overview

Create an empty file or update the timestamp of an existing file

### Command Parameters

```bash
-p|--path=string    Required parameter,File path. Example: /tmp/test.lock
```

### Example

```bash
acli system file touch create --path /tmp/acli_test_file
```

```bash
(Command executed successfully, no output)
```
